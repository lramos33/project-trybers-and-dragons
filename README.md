## 💻 Instalando o projeto

Clone o repositório:

```
git clone git@github.com:lramos33/project-trybers-and-dragons.git
```

Acesse a pasta do repositório

```
cd project-trybers-and-dragons
```

Instale as dependências:
```
npm install
```

Execute a aplicação
```
npm start
```
